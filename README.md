# Paws & Pixels: Dynamic Dog Image Viewer

Paws & Pixels is an interactive web app that loads dog breed information
and image galleries directly from the Dog API. Users can explore breeds
effortlessly while enjoying a sleek, auto-rotating slideshow experience.

## What This App Does
- Retrieves dog breed list from an external REST API
- Generates the breed dropdown dynamically
- Fetches all images of the selected breed
- Creates an animated slideshow with fade/zoom transitions
- Used React
